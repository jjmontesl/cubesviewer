CubesViewer

Visual data analysis and explore tool

Welcome to CubesViewer Explorer Demo. This is a visual, web-based tool application for exploring and analyzing data. The complete application has some extra features.

You can use the menu and interface to navigate this example dataset (composed of fictional sales of a web store during years 2012-2013 and also fictional web visits).

To begin, select a "Example view" or a "Cube" from the list on the left.
